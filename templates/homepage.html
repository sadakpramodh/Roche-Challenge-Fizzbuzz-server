FizzBuzz API

Overview

The FizzBuzz API is a RESTful web service that follows the classic FizzBuzz logic. Given number parameters, it returns the FizzBuzz sequence up to a limit, replacing multiples of the numbers with provided strings.

The API also exposes a statistics endpoint to retrieve usage analytics.

Endpoints

GET /fizzbuzz

Returns the FizzBuzz sequence based on provided request parameters.

Parameters

  • int1 - First number to substitute in sequence
  • int2 - Second number to substitute in sequence
  • str1 - String to replace multiples of int1
  • str2 - String to replace multiples of int2
  • limit - Upper limit of the sequence

Response

  • 200 OK on success with FizzBuzz sequence as a comma-delimited string
  • 400 Bad Request if invalid parameters provided

Example request

GET /fizzbuzz?int1=3&int2=5&str1=fizz&str2=buzz&limit=16

Example response

"1,2,fizz,4,buzz,fizz,7,8,fizz,buzz,11,fizz,13,14,fizzbuzz,16"

GET /stats

Retrieves statistics on the most frequent /fizzbuzz request.

Response

  • 200 OK with stats info as JSON
  • 500 Error on server error

Example response

{
        "int1": 3,
        "int2": 5,
        "str1": "fizz",
        "str2": "buzz"  
        "limit": 100,
        "count": 42  
        }

Architecture

The API is built with:

  • Flask - Python web framework
  • SQLite - Database for storage and analytics
  • Custom modules - Input validation, FizzBuzz logic implementation and unit tests
  • Templates - Basic homepage
  • Logging - Application logs for debugging

The code follows REST principles and OOP for modularity and extensibility. Input validation and error handling ensure stability.

Getting Started

Prerequisites: Python 3.x

Install:

pip install -r requirements.txt

Run:

python FizzBuzz_server.py

The server will now be live on localhost.
